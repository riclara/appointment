# Appointment Api Demo

>  A ES6 CRUD api
## Build Setup
```sh
# Run develop environment
$ git clone https://github.com/riclara/appointment.git # or clone your own fork
$ cd tasks
$ npm install
$ npm run dev
```
Your app should now be running on [localhost:8000](http://localhost:8000/).

## Tests
```sh
# Run Tests
$ npm test
```
